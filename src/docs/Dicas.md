1° Abrir o VSCODE
2° No terminal digitar: cd desktop
3° No terminal dentro de desktop: mkdir projeto
4° No terminal digitar: cd projeto
5° Dentro da pasta projeto: code .
6° No terminal da pasta projeto: touch index.html
7° No terminal: mkdir src
8° No terminal: cd src
9° No terminal: mkdir css
10° No terminal: mkdir js
11° No terminal: cd css
12° No terminal: touch estilo.css
13° No terminal: cd ..
14° No terminal: cd js
15° No terminal: touch script.js
16° No terminal: cd ..
17° No terminal: cd .. até chegar na raíz do projeto
18° NO TERMINAL: git init
19° Se estiver na FIAP: git config --global user.name "Matheus"
20° Se estiver na FIAP: gif config --global user.email matheus.tamataya@gmail.com
21° No terminal? git status
22° se estiver vermelho: git add .
22° se estiver verde: git commit -m "Mensagem do que você fez"
24° entrar no Github em repositorios, criar um novo repositorio, colocar o nome e deixar publico. clicar em criar repositorio (DICA DE OURO - NUNCA TRADUZA A PAGINA DO GITHUB)
25° No bloco que tiver 3 linhas:
    1° linha - Conecta o gitbash com o github
    2° linha - muda o nome master para main
    3° linha - pega tudo que estiver no projeto e sobe para o github
26° Caso você esteja em outro pc: no seu repositorio no botão verde (<CODE>), copie o link
27° Abra seu VSCODE no terminal ~/desktop : git clone (link), isso vai clonar seu projeto e pode continuar da onde parou
28° Se modificar seu código faça todo o processo de 
    git status 
    git add .
    git commit -m
    e para subir as alterações só digitar: git push
